# E-Learning Web Application

## Project Overview
A microservices-based e-learning marketplace that allows users to register as instructors, sell courses, and host interactive live classes.

## Technology Stack
- **Backend Architecture:** Microservices
- **Core Framework:** Java, Spring Boot, Spring Cloud
- **Frontend:** ReactJS
- **Database:** MySQL / PostgreSQL
- **Containerization:** Docker

## Learning Objectives
- Master microservices architecture concepts and inter-service communication.
- Implement real-world business logic for an online education platform.
- Follow clean code principles and standard Git workflow.
